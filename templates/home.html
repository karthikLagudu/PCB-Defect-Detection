How it works

From board image to annotated result

The application wraps a trained YOLOv5 object detector in a simple Flask workflow. It reads the uploaded image, runs model inference, draws the predicted regions, and returns the rendered inspection image.

  1. 01 Prepare the inputChoose a focused JPG, JPEG, or PNG board image up to 10 MB.
  2. 02 Run YOLOv5 inferenceThe model evaluates the image at size 415 using 0.50 confidence and 0.45 IoU thresholds.
  3. 03 Review the resultDetected regions are rendered on the image and returned as an annotated JPEG.
